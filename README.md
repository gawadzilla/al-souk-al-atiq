# Al-Souk Al-Atiq Restaurant Menu

A beautifully crafted, elegant HTML menu website for "Al-Souk Al-Atiq" (مطعم السوق العتيق), an authentic Lebanese restaurant located in Soufar, Mafrak El Chouf.

## Overview

This is a standalone, responsive HTML/CSS webpage that serves as a digital menu for a Lebanese restaurant. The design features a luxurious, vintage aesthetic with gold accents, Arabic typography, and an elegant presentation of menu items.

## Features

- **Responsive Design** – Adapts seamlessly to desktop, tablet, and mobile screens
- **Bilingual Presentation** – Menu items displayed with both English and Arabic headers
- **Animated Entrance** – Elegant intro animation with logo reveal
- **Scroll Animations** – Content fades in smoothly as you scroll
- **Complete Menu Categories**:
  - Shawarma (Chicken, Beef, Turkey, Signature options)
  - Sandwiches (Tawouq, Chicken Breast, Btata)
  - Special Menu (Chicken Tawook Meal, Shawarma Aripa)
  - Drinks (Pepsi, Water, Ayran)
  - Combo upgrades and side dishes

## Technologies Used

- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- Google Fonts (Cormorant Garamond, Cinzel, Amiri)
- Vanilla JavaScript (scroll animations, intro fade-out)
